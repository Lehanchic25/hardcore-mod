# Hardcore Mod
 A single JS script that makes game a lot harder.
 
 Thx Sputnic for optimization and the idea of changing statistics using js script
 
 This mod: 
 1. Makes all blocks cost 10 times more
 2. Makes hp of blocks 10 times less
 3. Makes itemCapacity in all Cores 10 times more (so you could place some blocks)
 4. Makes buildCostMultiplier = 10 (so you can place blocks and not wait for eternity)
 5. Idk makes early game a lil bit harder (drillTime of mechanicalDrill is *= 1.25)
